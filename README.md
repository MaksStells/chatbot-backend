# Chatbot with Groq - Leo

First install node.js on nodejs.org.
Download files and put them in a folder.
Open code in VSCode on administrator. 
Do CTRL + ' to open terminial (or just find it) and type "node -v", it should show a version number.

GROQ KEY
Have an account on Groq https://console.groq.com/keys
Then make a new API key, and copy that key in the space in the .env file.

Then in the terminal, type "node server.js" and it should say 'UniBot server running at http://localhost:3000'
Open that in your browser and it might work.
